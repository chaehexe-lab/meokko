# 머꼬

AI 기반 고객데이터 수집·세분화 및 꼬모냉장고 자동화 마케팅 시스템 구축 프로젝트

## 프로젝트 목표

꼬모냉장고의 국내 판매량 증가를 목표로, 시장 반응과 고객 데이터를 분석하여 핵심 타겟층을 도출하고 타겟별 마케팅 방향성을 제안하는 시스템을 구축한다.

프로젝트는 두 단계로 진행된다.

1. 시장 반응, 구매 저해 요인, 핵심 타겟층을 분석하는 STEP1 대시보드 개발
2. STEP1 분석 결과를 바탕으로 타겟별 홍보 채널과 마케팅 콘텐츠를 추천하는 STEP2 대시보드 개발

## 주차별 계획

| 주차 | 주요 계획 |
| --- | --- |
| 1주차 | 고모텍 현장 방문, 프로젝트 목표 설정, 전체 방향성 수립 |
| 2주차 | 꼬모냉장고 제품 분석, 국내 소형 냉장고 시장 조사, STEP1 대시보드 초안 설계 |
| 3~4주차 | 데이터 수집 후 시장 반응, 구매 저해 요인, 핵심 타겟층 분석 기능 개발 |
| 5주차 | 홍보 채널로 활용할 플랫폼 분석 및 STEP2 대시보드 초안 설계 |
| 6~7주차 | 플랫폼별 홍보 방식 구체화 및 AI 콘텐츠 추천 시스템 개발 |
| 8주차 | 프로토타입 시연, 최종 발표자료 제작, 운영 가이드 정리 |
