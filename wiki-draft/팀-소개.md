# 팀 소개

## 팀명

머꼬

## 팀원

| 이름 | 역할 |
| --- | --- |
| 이채희 | 대장 / 프로젝트 총괄 |
| 김은결 | 데이터 수집 및 시장 데이터 분석 |
| 조원경 | 고객 데이터 분석 및 인사이트 도출 |
| 천지원 | 대시보드 기획 및 콘텐츠 전략 정리 |

## 역할 설명

### 이채희

프로젝트 전체 방향을 관리하고, 일정 조율과 결과물 정리를 담당합니다.

### 김은결

검색량, 시장 트렌드, 경쟁 제품 관련 데이터를 수집하고 시장 데이터 분석을 담당합니다.

### 조원경

리뷰, 문의, CRM 데이터를 바탕으로 구매 동기와 구매 저해 요인을 분석합니다.

### 천지원

STEP1·STEP2 대시보드 구성과 타겟별 콘텐츠 전략 정리를 담당합니다.
