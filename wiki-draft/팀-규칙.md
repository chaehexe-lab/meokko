# 팀 규칙

## 회의 일정

- 정기 회의: 매주 월요일 오후 7시
- 멘토링: 매주 수요일 12시

## 기본 규칙

- 회의 전까지 각자 진행한 내용을 간단히 정리합니다.
- 막히는 부분은 혼자 오래 고민하지 않고 팀원에게 공유합니다.
- 일정 변경이 필요할 경우 가능한 빨리 팀원들에게 알립니다.
- 주요 결정 사항은 회의록에 남깁니다.
- 작업 내용은 GitHub와 Notion을 기준으로 공유합니다.

## GitHub 사용 규칙

- 작업 내용은 가능한 한 작은 단위로 나누어 커밋합니다.
- 커밋 메시지는 작업 내용을 알아볼 수 있게 작성합니다.
- 중요한 변경 사항은 팀원들과 공유한 뒤 반영합니다.
- README와 Wiki는 프로젝트 진행 상황에 맞게 계속 업데이트합니다.
